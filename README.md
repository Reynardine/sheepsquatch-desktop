# Getting Started:
#### !!!IMPORTANT!!!
You have to edit the tint2rc AND compton.conf for your needs! (wonderfully made by hazuki over at gnome-look <3)<br/>
The Default icons are set to:<br/>
<br/>
Terminal: lxterminal<br/>
Files: pcmanfm<br/>
Browser: Brave<br/>
